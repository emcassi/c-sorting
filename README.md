<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* Install <a href="https://wiki.libsdl.org/SDL2/Installation">SDL2</a>

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/emcassi/c-sorting.git
   ```
2. Compile and run the project
   ```sh
   make # Compile
   ./csorting # Run

   # # OR # # 

    sh run.sh # Compile and run
   ```
